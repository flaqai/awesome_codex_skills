## Perché questa lista

- I Codex Skill sono più utili quando gli utenti trovano rapidamente il workflow giusto.
- Questo progetto cura Skill pratici, li raggruppa per use case e registra link, segnali di popolarità e note di preinstallazione.

## Sfoglia per categoria

- [Strumenti Agent](#strumenti-agent) - Skill di base che aiutano Codex a trovare, combinare o creare altri Skill.
- [Sviluppo e frontend](#sviluppo-e-frontend) - Skill per frontend, UI, deploy, comprensione dei repository e workflow tecnici.
- [Creatività e media](#creatività-e-media) - Skill creativi per immagini, audio, video, visual marketing e produzione media.
- [Automazione browser e test](#automazione-browser-e-test) - Skill per controllo browser, screenshot, controlli UI e validazione delle interazioni.
- [Documenti e presentazioni](#documenti-e-presentazioni) - Skill per PDF, slide, report, proposte e documenti strutturati.
- [Dati e visualizzazione](#dati-e-visualizzazione) - Skill per grafici, spiegazioni visuali, analisi e narrazioni basate sui dati.
- [Marketing e crescita](#marketing-e-crescita) - Skill per SEO, copy, asset di lancio, campagne e workflow di crescita.
