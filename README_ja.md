## このリストについて

- Codex Skill は、ユーザーが目的に合うワークフローを素早く見つけられるほど価値が高まります。
- このプロジェクトは実用的な Skill を用途別に整理し、リンク、人気指標、プリインストール時の注意点を記録します。

## カテゴリ別に見る

- [エージェントツール](#エージェントツール) - Codex が他の Skill を見つけ、組み合わせ、作成するのを助ける基盤スキル。
- [コーディングとフロントエンド](#コーディングとフロントエンド) - フロントエンド、UI、デプロイ、リポジトリ理解、エンジニアリングワークフロー向けの Skill。
- [クリエイティブとメディア](#クリエイティブとメディア) - 画像、音声、動画、マーケティングビジュアル、メディア制作向けのクリエイティブ Skill。
- [ブラウザ自動化とテスト](#ブラウザ自動化とテスト) - ブラウザ操作、スクリーンショット、UI チェック、操作検証のための Skill。
- [ドキュメントとプレゼン](#ドキュメントとプレゼン) - PDF、スライド、レポート、提案書、構造化文書向けの Skill。
- [データと可視化](#データと可視化) - チャート、視覚的説明、分析、データストーリー向けの Skill。
- [マーケティングと成長](#マーケティングと成長) - SEO、コピー、ローンチ素材、キャンペーン、成長ワークフロー向けの Skill。
- [業界別スキルとコンプライアンス](#業界別スキルとコンプライアンス) - 医療、アクセシビリティ、セキュリティ、ローカライズなど領域特化の Skill。
