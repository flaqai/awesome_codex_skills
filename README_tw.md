## 為什麼做這個列表

- Codex Skill 的價值在於使用者能快速找到適合當前任務的工作流。
- 本專案精選實用 Skill，按用途分類，並記錄來源連結、熱度依據和預裝注意事項。

## 按分類瀏覽

- [Agent 工具鏈](#agent-工具鏈) - 幫助 Codex 查找、組合或建立其他 Skill 的基礎能力。
- [程式開發與前端](#程式開發與前端) - 前端、UI、部署、程式庫理解和工程工作流相關 Skill。
- [創作與媒體](#創作與媒體) - 涵蓋圖片、音訊、影片、行銷視覺和媒體製作的創作類 Skill。
- [瀏覽器自動化與測試](#瀏覽器自動化與測試) - 用於瀏覽器控制、截圖、UI 檢查和互動驗證的 Skill。
- [文件與簡報](#文件與簡報) - 用於 PDF、投影片、報告、提案和結構化文件的 Skill。
- [資料與視覺化](#資料與視覺化) - 用於圖表、視覺化說明、資料分析和資料敘事的 Skill。
- [行銷與成長](#行銷與成長) - 用於 SEO、文案、發布素材、活動和成長工作流的 Skill。
- [產業能力與合規](#產業能力與合規) - 醫療、可訪問性、安全、本地化和其他垂直領域 Skill。

- 連結必須可訪問，不能是 legacy 錯誤頁。

- 條目需要有清晰用途和來源平台。
- 優先收錄高使用量、高下載量、官方精選或高 stars 條目。
- 高風險領域和外部服務依賴必須明確標註。
