## 为什么做这个列表

- Codex Skill 的价值在于用户能快速找到适合当前任务的工作流。
- 本项目会精选实用 Skill，按用途分类，并记录来源链接、热度依据和预装注意事项。

## 按分类浏览

- [Agent 工具链](#agent-工具链) - 帮助 Codex 查找、组合或创建其他 Skill 的基础能力。
- [编程与前端](#编程与前端) - 前端、UI、部署、仓库理解和工程工作流相关 Skill。
- [艺术创作与媒体](#艺术创作与媒体) - 覆盖图片、音频、视频、营销视觉和媒体生产的创作类 Skill。
- [浏览器自动化与测试](#浏览器自动化与测试) - 用于浏览器控制、截图、UI 检查和交互验证的 Skill。
- [文档与演示](#文档与演示) - 用于 PDF、幻灯片、报告、提案和结构化文档的 Skill。
- [数据与可视化](#数据与可视化) - 用于图表、可视化说明、数据分析和数据叙事的 Skill。
- [营销与增长](#营销与增长) - 用于 SEO、文案、发布素材、活动和增长工作流的 Skill。
- [行业能力与合规](#行业能力与合规) - 医疗、可访问性、安全、本地化和其他行业垂直 Skill。

## 收录标准

- 链接必须可访问，不能是 legacy 错误页。
- Skill 应该对 Codex 有用，或容易改造成 Codex 工作流。
- 条目需要有清晰用途和来源平台。
- 优先收录高使用量、高下载量、官方精选或高 stars 条目。
- 高风险领域和外部服务依赖必须明确标注。
